# Booth WooCommerce Theme

## Project Overview

Booth is a WooCommerce-based theme designed to integrate seamlessly with the Elementor page builder, offering a flexible and customizable solution for creating engaging online stores. Developed with modern web technologies and practices, Booth aims to provide a robust platform for e-commerce websites with a focus on usability and performance.

## Features

- **WooCommerce Integration**: Full compatibility with WooCommerce for a complete e-commerce solution.
- **Elementor Page Builder**: Leverages the power of Elementor for easy drag-and-drop page customization.
- **Responsive Design**: Ensures a seamless experience across all devices.
- **Lightbox Gallery**: Built-in support for image lightboxes using Magnific Popup.
- **Customizable Sections**: Utilise WordPress Customizer for tailoring specific sections to your needs.

## Getting Started

### Installation

To begin using the Booth WooCommerce theme, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/tim-green/booth-woocommerce-theme.git
   ```

2. **Navigate to the Project Directory**:
   ```bash
   cd booth-woocommerce-theme
   ```

3. **Install Node Dependencies**:
   Ensure you have Node.js installed, then run:
   ```bash
   npm install
   ```

### Usage

1. **Build and Watch for Changes**:
   Utilise Grunt to compile assets and watch for changes:
   ```bash
   npm run watch
   ```

2. **Activate the Theme**:
   - Upload the theme to your WordPress installation.
   - Activate it via the WordPress admin panel under Appearance > Themes.

3. **Customize Your Site**:
   - Use the WordPress Customizer to adjust settings and layout.
   - Leverage Elementor to build and customise pages.
